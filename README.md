# Sentra

**Sentra** is a local-first, offline-capable, deterministic agentic automation platform.

## Features

- Improved security model
- Enhanced modularity
- Better performance optimizations
- Advanced memory management
- Improved cross-platform compatibility
- Enhanced plugin system
- Better error handling and recovery

## Architecture

Sentra builds upon the foundation of OpenClaw with several improvements:

- Modular core design
- Pluggable transport layer
- Enhanced security sandboxing
- Optimized resource management
- Advanced AI model switching
- Improved persistence layer
